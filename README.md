# Restaurant Business Generator

This project is an AI-powered application that generates restaurant names and menu ideas based on selected cuisine.

## 🚀 Features
- Generate creative restaurant names
- Suggest menu items based on cuisine
- Built using LangChain and OpenAI

## 🛠️ Tech Stack
- Python
- LangChain
- OpenAI API
- Gradio

## 📂 Project Structure
- `your_notebook.ipynb` – Main implementation

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Add your OpenAI API key
3. Run all cells

## 💡 Learning Outcome
This project demonstrates:
- Use of LangChain chains and agents
- Prompt-based generation
- Building simple AI applications
